# React Notes App 📝

A modern Notes App built with React, Tailwind CSS, and localStorage.

## Features

- Add Notes
- Delete Notes
- Notes saved in localStorage
- Responsive Design
- Modern Dark UI

##  Tech Stack

- React
- Tailwind CSS
- JavaScript
- Vite



##  Installation

```bash
npm install
npm run dev
```
